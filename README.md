# Location Sample
Finding location with GPS or Network Android project

## Requirements
```bash
Android Studio Artic Fox 2020.3.1 Canary 10 or higher,
Gradle version: 7.0.0 Alpha 10,
SDK API 30,
Platform Tools: 30.0.3,
AndroidX libraries,
Kotlin or Java
```
